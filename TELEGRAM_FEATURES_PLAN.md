# Vidnovagram — План реалізації Telegram фіч

## Статус фіч

### ✅ Вже реалізовано
- Текстові повідомлення
- Фото (тамбнейл + повне зображення)
- Відео (autoplay + controls)
- Кружки / video_note (круглі)
- Голосові повідомлення (аудіоплеєр)
- Документи (PDF в браузері, інші — збереження)
- Відповідь з цитатою (reply with quote)
- Пересилання (forward, multi-select)
- Редагування своїх повідомлень
- Реакції (6 швидких emoji + аватарки)
- Статус прочитання (✓ / ✓✓)
- Typing індикатор (набирає повідомлення...)
- Запис голосових і відео-кружків
- Drag & drop файлів
- Прев'ю посилань (link preview з картинкою)
- Контекстне меню (ПКМ: відповісти, копіювати, переслати, редагувати)
- Дата-розділювачі між днями
- Десктопні сповіщення + звук
- Видалені повідомлення (відображення "видалено", чорний ящик)

### 🔲 Не реалізовано — ПЛАН
